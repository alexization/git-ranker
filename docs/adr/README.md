# ADR Index

Architecture Decision Records (ADRs) capture durable design decisions.

## Naming
- `ADR-0001-<short-title>.md`

## When to Write ADR
1. Dependency direction or layer policy changes.
2. Cross-cutting patterns (auth, logging, error model) change.
3. Major build/deploy/observability architecture changes.
